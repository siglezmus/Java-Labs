create git repo:
git init

check commit:
git status

add files to commit:
git add . // all files in repo
git add -p <file> // one file

some stuff to test status

change branch
git checkout <branch>

create branch
git create branch

merge targeted branch into current
git merge <branch>

rebase current branch into targeted
git rebase <branch>

tool for solving merge conflicts
git mergetool

checking existing remotes:
git remote -v

adding existing remote:
git remote add <name>origin <URL>https://github.com/siglezmus/Java-Labs.git

pushing to git:
git push origin <branch>master

pulling from git:
git pull origin master
